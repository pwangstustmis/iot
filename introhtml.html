網頁標籤

本單元僅介紹常用HTML標籤及其屬性，詳細的介紹，請參考W3Schools HTML。

網頁檔架構與標籤介紹

網頁檔案內容架構如下： 


  <html>
    <head>
      <!--網頁註解格式(設定內嵌utf-8編碼) -->
      <meta charset="utf-8">

      <!--網頁註解格式(設定內嵌JavaScript) -->
      <script>

      </script>

      <!--網頁註解格式(設定內嵌CSS樣式) -->
      <style>

      </style>
    </head>
    <body>
      <!--網頁顯示內容 -->
    </body>
  </html>

檔頭區塊(head)用於設定詮釋資料包含編碼設定(charset)、抬頭設定(title)、JavaScript設定與樣式CSS設定等，所有的顯示內容請至於body區塊。如同檔案內容架構所示，HTML標籤格式為<標籤名稱>，顯示內容至於開始標籤與結束標籤之間，結束標籤為</標籤名稱>，但是也有少數標籤為自我結束標籤(self closing tag)其格式為<標籤名稱/>。標籤名稱一般為英文縮寫，例如標題英文為(heading)，因此HTML的六種預設標題標籤名稱為h1到h6。

HTML標籤除了標籤名稱之外，最重要的是標籤屬性(attribute)設定，網頁內容利用標籤名稱指定內容的類別，之後利用標籤屬性來設定內容的細項，標籤屬性格式為<標籤名稱 屬性名稱="屬性值" >，以常用自我結束圖片標籤img為例，<img src="圖片檔案位置" >中屬性src用於設定圖片檔案位置，了解HTML標籤格式後，接下來介紹常用HTML標籤。

常用標籤-檔頭位置標籤h

標籤h預設用於設定標題，有六種預設大小(h1到h6)，標題標籤預設會跳行，使用者可以搭配CSS改變標題樣式。範例結果：

h1

,

h2

,

h3

HTML原始碼：


<h1>h1</h1>
<h2>h2</h2>
<h3>h2</h3>
常用標籤-段落p

標籤p用於設定段落內容，每個段落有預設間距，使用者可以搭配CSS改變段落樣式。段落內跳行請使用分行<br/>標籤。範例結果：

南臺科技大學
資訊管理系

HTML原始碼：


<p>南臺科技大學<br/>資訊管理系</p>
常用標籤-圖片img

標籤img用於網頁圖片，常用屬性包含src與alt，src用於設定圖片檔案名稱，圖片檔位置則相對網頁本身，當無法取得圖片時，alt用於設定顯示文字，img標籤雖可使用width(寬)或height(高)來設定圖片大小，一般仍建議使用樣式來進行控制。範例結果：

HTML原始碼：


<p><img src="imgs/introhtml/imgtag.png" alt="南臺校徽"/>
常用標籤-超連結a

標籤a用於設定超連結，常用屬性包含href與target，href用於連結對象，如果連結對象為網站內的資源，請使用相對檔案位置，如果連結對象為網站外部資源，請使用絕對位置(必須指定全部網址)，href值如果設為 # 則代標空連結。target用於設定連結內容顯示頁面，當設定為 _blank 時，瀏覽器會開新的頁面顯示連結內容，不指定則內容會顯示在現有的頁面。範例結果：

內部連結南臺校徽, 外部連結南臺校徽, 新頁面內部連結南臺校徽

HTML原始碼：


<a href="imgs/introhtml/imgtag.png">內部連結南臺校徽</a>
<a href="https://www.stust.edu.tw/tc/images/about/logo.png">外部連結南臺校徽</a>
<a href="imgs/introhtml/imgtag.png" target="_blank">新頁面內部連結南臺校徽</a>
常用標籤-表格table

標籤table用於設定表格，tr 標籤代表表格列，tr 標籤個數代表表格列個數，儲存格使用 th 與 td 標籤，th 用於設定表格標題，td 用於設定一般儲存格，表格也提供合併儲存格功能，透過 th 與 td 標籤中 rowspan 與 colspan 屬性設定合併列或合併欄，表格格式一般使用樣式進行設定。範例結果：

標題1 標題2
內容1 內容2

HTML原始碼：


<table class="table table-bordered border-primary" style="width: fit-content;">
  <tr>
    <th>標題1</th>
    <th>標題2</th>
  </tr>
  <tr>
    <td>內容1</td>
    <td>內容2</td>
  </tr>
</table>
合併列標題 內容1
內容2

HTML原始碼：


<table class="table table-bordered border-primary" style="width: fit-content;">
  <tr>
    <th rowspan="2">合併列標題1</th>
    <td>內容1</td>
  </tr>
  <tr>
    <td>內容2</td>
  </tr>
</table>
合併欄標題
內容1 內容2

HTML原始碼：


  <table class="table table-bordered border-primary" style="width: fit-content;">
    <tr>
      <th colspan="2">合併欄標題1</th>
    </tr>
    <tr>
      <td>內容1</td>
      <td>內容2</td>
    </tr>
  </table>
常用標籤-項目ul

標籤ul用於設定無編號項目，ul標籤中每個項目使用li標籤，可藉由樣式設定不同樣式符號等。範例結果：

  • 項目1
  • 項目2
  • 項目3

HTML原始碼：


<ul>
  <li>項目1</li>
  <li>項目2</li>
  <li>項目3</li>
</ul>
常用標籤-符號ol

標籤ol用於設定編號項目，ol標籤中每個項目使用li標籤，可藉由樣式設定不同樣式編號等。範例結果：

  1. 編號1
  2. 編號2
  3. 編號3

HTML原始碼：


<ol>
  <li>編號1</li>
  <li>編號2</li>
  <li>編號3</li>
</ol>
方塊區域標籤-div與span

標籤div與span用於網頁方形區域位置設定，不同div區塊會跳行，span預設在同一行，例如在同一行中的某些內容要改變樣式，則可使用span。範例結果：

div方塊大小150 x 150紅色1點邊框

HTML原始碼：


<div style="width:150px;height:150px;border:#f00 1px solid;">div方塊大小150 x 150紅色1點邊框</div>

段落文字要顯示紅色部分

HTML原始碼：


<p>段落文字要顯示<span style="color:#f00">紅色</span>部分</p>

連結外部樣式-link

用於連結外部檔案，常用屬性為rel與href，rel用於設定網頁檔與連接檔案關係，最常用rel設定為stylesheet，用於設定外部連接檔案為樣式檔，href用於設定連接檔案位置。以下範例用於連結最為普遍樣式庫Bootstrap。

HTML原始碼：


<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.0/dist/css/bootstrap.min.css" 
integrity="sha384-KyZXEAg3QhqLMpG8r+8fhAXLRk2vvoC2f3B09zVXn8CA5QIVfZOJ3BCsw2P0p/We" crossorigin="anonymous">
聲音-audio

audio標籤用於撥放聲音檔，支援格式包含MP3、WAV與OGG，請注意Apple Safari瀏覽器不支援OGG格式，audio常用屬性包含 controls(用於設定顯示控制按鈕)、autoplay(用於設定自動撥放)、muted(靜音)，audio標籤會包含source標籤用於設定聲音檔來源(src)與格式(type)，請注意：可以設定多種格式，audio會對音源檔由上而下檢查是否可以撥放。

HTML原始碼：


<audio controls autoplay muted>
  <source src="imgs/introhtml/audiosample.ogg" type="audio/ogg">
  <source src="imgs/introhtml/audiosample.mp3" type="audio/mp3">
  瀏覽器不支援音樂撥放
</audio>
影片-video

video標籤用於撥放影片檔，支援格式包含MP4、WebM與OGG，請注意Apple Safari瀏覽器不支援OGG格式，video常用屬性包含 controls(用於設定顯示控制按鈕)、autoplay(用於設定自動撥放)、muted(靜音)、width(撥放器寬)、height(撥放器高)、poster(載入時顯示圖片檔)，video標籤會包含source標籤用於設定影片檔來源(src)與格式(type)，請注意：可以設定多種格式，video會對影片檔由上而下檢查是否可以撥放。

HTML原始碼：


<video controls muted>
  <source src="imgs/introhtml/videosample.ogg" type="video/ogg">
  <source src="imgs/introhtml/videosample.mp4" type="video/mp4">
  瀏覽器不支援影片撥放
</video>

表單輸入標籤input是表單中使用最為頻繁的標籤，用於資料輸入，藉由type屬性設定text(文字)、button(按鈕)、color(顏色選取)、date(日期選取)、datetime-local(日期及時間選擇)、email(電子郵件輸入)、file(檔案選擇)、hidden(隱藏表單欄位)、image(圖片按鈕)、month(月份選擇)、number(數字選擇器)、password(密碼輸入)、range(範圍選擇器)、submit(送出按鈕)、tel(電話號碼)、time(時間選擇器)、week(月份選擇器)。屬性依據不同類型而有所不同，請參考MDN Wen DOcs Input element，如果input標籤一定要輸入則含屬性required ，要傳送給後端程式表單欄位請設定name屬性用於後端程式對應個別欄位，input標籤一般會搭配label標籤用於設定欄位標籤文字。
